== Description ==
Tab extension for Kirki Customizer Framework.

== Installation ==
There are two ways to install Kirki Pro's `control-tab`:
- Either put the `control-tab` folder into `wp-content/plugins/` directory
- OR if you would like to just embed it inside of your theme or plugin, then just require the main file (adjust it based on your path): `require_once 'path-to/control-tab/control-tab.php';`

== Frequently Asked Questions ==

= Why is there no automatic update? =
Because Kirki Pro can be embeded in theme / plugins, then they need to be updated manually.
