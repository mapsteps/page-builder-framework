== Description ==
Headlines & divider extension for Kirki Customizer Framework.

== Installation ==
There are two ways to install `kirki-pro-headline-divider`:
- Either put the `kirki-pro-headline-divider` folder into `wp-content/plugins/` directory
- OR if you would like to just embed it inside of your theme or plugin, then you'll need to require the autoload (adjust it based on your path): `require_once 'path/kirki-pro-headline-divider/vendor/autoload.php';` and then run the init: `new \Kirki\Pro\HeadlineDivider\Init();`

== Frequently Asked Questions ==

= Why is there no automatic update? =
Because Kirki Pro can be embeded in theme / plugins, then they need to be updated manually.
